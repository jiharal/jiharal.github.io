Anagram

December 18, 2019

Anagram teori

Anagram adalah salah satu jenis permainan kata yang huruf-huruf di kata awal biasanya diacak untuk membentuk kata lain atau sebuah kalimat. Anagram sering dipakai sebagai kode. Permainan yang menggunakan anagram antara lain adalah scrabble dan boggle.Wikipedia

Studi kasus

Dalam tantangan ini, Anda akan diberikan string. Anda harus membaginya menjadi dua substring yang berdekatan, kemudian menentukan jumlah karakter minimum yang harus diubah untuk membuat dua substring menjadi anagram satu sama lain.

Misalnya, mengingat string ‘abccde’, Anda akan memecahnya menjadi dua bagian: ‘abc’ dan ‘cde’. Perhatikan bahwa semua huruf telah digunakan, substring bersebelahan dan panjangnya sama. Sekarang Anda dapat mengubah ‘a’ dan ‘b’ di substring pertama menjadi ‘d’ dan ‘e’ untuk memiliki ‘dec’ dan ‘cde’ yang merupakan anagram. Diperlukan dua perubahan.

Buatlah fungsi anagram untuk mengembalikan jumlah karakter minimum untuk membuat kata-kata anagram, kembalikan -1 jika tidak memungkinkan.


package main

import (
	"fmt"
	"strings"
)

func main() {
	fmt.Println(Anagram("abccba"))
	fmt.Println(Anagram("aaabbb"))
}

func Anagram(s string) int32 {
	val1 := s[:len(s)/2]
	val2 := s[len(s)/2:]
	var count int32 = 0
	if len(s)%2 != 0 {
		return -1
	} else if string(val1) == string(val2) {
		return 0
	}

	for _, a := range val1 {
		index := strings.Index(val2, string(a))
		if index == -1 {
			count = count + 1
		} else {
			val2 = fmt.Sprintf("%s%s", val2[:index], val2[index+1:])
		}
	}
	return count
}
