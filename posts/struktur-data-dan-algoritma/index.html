Struktur Data dan Algoritma

December 18, 2019

Algoritma adalah prosedur langkah demi langkah, yang mendefinisikan serangkaian instruksi yang akan dieksekusi dalam urutan tertentu untuk mendapatkan output yang diinginkan.

Struktur data adalah sebuah cara sistematik untuk mengorganisir data agar data dapat dengan digunakan dengan mudah dan efisien. Ada dua istilah dasar dalam struktur data diantaranya Interface and Implementation. Setiap struktur data mempunyai interface. Interface merepresentasikan serangkaian operasi-operasi yang didukung oleh struktur data itu sendiri. Interface menyediakan daftar dari operasi yang didukung, type dan parameter yang dapat di terima dan dikembalikan oleh jenis operasi itu sendiri. Implementasi dari struktur data memberikan representasi dari struktur data juga menyediakan definisi-definisi dari algoritma yang digunakan dalam operasi struktur data.

  • Karakteristik dari struktur data

    • Correctness implementasi struktur data yang benar harus mengimplementasikan interface dengan benar.
    • Time Complexity Waktu pelaksanaan dari sebuah operasi harus sekecil mungkin.
    • Space Complexity penggunaan memori/space pada setiap operasi struktur data harus sekecil mungkin.

  • Mengapa struktur data dibutuhkan

    Kita mungkin kenal dengan produk facebook, google, amazon, alibaba produk dari mereka adalah produk aplikasi yang sangat kompleks dan kaya akan data, tiga masalah umum yang mungkin terjadi apa aplikasi tersebut ialah Pencarian data,Kecepatan eksekusi dari sebuah CPU untuk mengolah data, dan jutaan request yang terjadi hampir setiap detik pada aplikasi sehingga membuat aplikasi semakin melambat. Dengan struktur data yang baik dan benar masalah tersebut bisa di atasi dengan baik. Data dapat diatur dalam struktur data sedemikian rupa sehingga semua item mungkin tidak diperlukan untuk dicari, dan data yang diperlukan dapat dicari hampir secara instan.

  • Struktur data dasar

    • Type data adalah sebuah cara untuk mengklarifikasikan berbagai jenis data seperti integer, float, string dll. Yang digunakan untuk menentukan nilai-nilai dengan jenis data yang sesuai. Ada dua jenis type data diantaranya Built-in Data Type dan Derived Data Type. Jenis type data Built-in adalah jenis type data yang didukung oleh suatu bahasa dikenal dengan type data bawaan umunya seperti int, string, bool, float. Derived Data Type ialah type data yang yang di implementasikan secara independen karena dapat di implementasikan dengan satu cara atau cara lain. Contoh List, Array, Stack, dan Queue.
