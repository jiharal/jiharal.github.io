Palindrome Palindrom adalah sebuah kata, frasa, angka maupun susunan lainnya yang dapat dibaca dengan sama baik dari depan maupun belakang (spasi antara huruf-huruf biasanya diperbolehkan). Kata “palindrom” berasal dari bahasa Yunani: palin (πάλιν; “lagi”) and dromos (δρóμος; “arah”).wikipedia package main import ( "fmt" ) func main() { fmt.Println(palindrome("Hello")) fmt.Println(palindrome("abcba")) fmt.Println(palindrome("coba")) fmt.Println(palindrome("kamak")) } func palindrome(s string) bool { for i:=0; i<len(s)/2; i++ { if s[i] != s[len(s)-i-1] { return false } } return true } PlayGolang
