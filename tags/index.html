Struktur Data dan Algoritma

Algoritma adalah prosedur langkah demi langkah, yang mendefinisikan serangkaian instruksi yang akan dieksekusi dalam urutan tertentu untuk mendapatkan output yang diinginkan. Struktur data adalah sebuah cara sistematik untuk mengorganisir data agar data dapat dengan digunakan dengan mudah dan efisien. Ada dua istilah dasar dalam struktur data diantaranya Interface and Implementation. Setiap struktur data mempunyai interface. Interface merepresentasikan serangkaian operasi-operasi yang didukung oleh struktur data itu sendiri. Interface menyediakan daftar dari operasi yang didukung, type dan parameter yang dapat di terima dan dikembalikan oleh jenis operasi itu sendiri.

Kata “palindrom” berasal dari bahasa Yunani: palin (πάλιν; “lagi”) and dromos (δρóμος; “arah”).wikipedia package main import ( "fmt" ) func main() { fmt.Println(palindrome("Hello")) fmt.Println(palindrome("abcba")) fmt.Println(palindrome("coba")) fmt.Println(palindrome("kamak")) } func palindrome(s string) bool { for i:=0; i<len(s)/2; i++ { if s[i] != s[len(s)-i-1] { return false } } return true } PlayGolang

Object Oriented Programming (OOP)

Pemrograman berorentasi objek merupakan sebuah paradigma pemrograman yang berorientasi pada objek. Inti dari object-oriented programming yaitu membuat sebuah objek di dalam kode program dan memiliki data berupa atribut dan method tertentu. Misal kita memiliki objek Mobil, Dalam OOP Mobil adalah sebuah benda/objek yang memiliki sifat tertentu seperti warna, jumlah pintu, dan sejenisnya.
